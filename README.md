
# 📊 Subscriber Analysis & View Growth Dashboard

## 📌 Project Overview
This data analytics project analyzes subscriber growth patterns, video view trends, and audience engagement metrics to derive actionable insights for content optimization. The project demonstrates end-to-end analytics from data extraction to interactive dashboard creation.

## 🛠️ Tools & Technologies Used
- **Python** (Pandas, Matplotlib, Seaborn) - Data cleaning & analysis
- **SQL** - Data querying and transformation
- **Excel** - Interactive dashboard creation
- **Jupyter Notebook** - Exploratory data analysis

## 📈 Key Insights
- Identified peak subscriber growth periods correlated with specific content types
- Discovered optimal video length for maximum audience retention
- Found content patterns that improve engagement rate by 25%
- Determined best posting schedule for subscriber acquisition

## 📊 Dashboard Features
- **KPI Cards** - Total subscribers, views, engagement rate
- **Growth Trends** - Daily/Weekly/Monthly subscriber acquisition
- **Content Performance Matrix** - Views vs Engagement analysis
- **Geographic Distribution** - Audience locations
- **Retention Analysis** - Watch time patterns

## 💡 Business Impact
- **Content Strategy** - Data-driven recommendations for content calendar
- **Audience Targeting** - Identified high-value subscriber segments
- **Performance Monitoring** - Real-time KPI tracking system
- **ROI Improvement** - 30% increase in targeted content effectiveness

## 🚀 How to Use This Project
1. Clone the repository
2. Open Jupyter notebook for analysis steps
3. Review Excel dashboard for interactive exploration
4. Check SQL queries for data transformation logic
